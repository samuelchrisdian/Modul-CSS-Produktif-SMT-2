Font Style

Cascading Style Sheet (CSS) merupakan aturan untuk mengatur beberapa komponen dalam sebuah web sehingga akan lebih terstruktur dan seragam. CSS bukan merupakan bahasa pemograman.

  • text-align: digunakan untuk mengatur perataan teks, value yang dapat digunakan yaitu left, center, right
  • color: digunakan untuk memberikan watna pada teks
  • letter-spacing: mengatur jarak antar karakter
  • word-spacing: mengatur jarak antar kata
